# Projet Sudoku

## Description
Ce projet est un jeu de Sudoku réalisé en HTML, CSS et JavaScript dans le cadre d’un projet de cours.

Le joueur peut remplir une grille de Sudoku en respectant les règles classiques :
- Chaque chiffre de 1 à 9 doit apparaître une seule fois par ligne
- Une seule fois par colonne
- Une seule fois par carré 3x3

## Technologies utilisées
- HTML
- CSS
- JavaScript

## Lancement du projet
Pour lancer le jeu :
1. Télécharger le projet
2. Ouvrir le fichier `index.html` dans un navigateur
